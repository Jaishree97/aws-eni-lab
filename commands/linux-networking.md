# Linux Networking Commands

## Switch to Root User

```bash
sudo su
```

## Check Current User

```bash
whoami
```

## List Files

```bash
ls
```

## Show Current Directory

```bash
pwd
```

## Check Server Hostname

```bash
hostname
```

## Display IP Address

```bash
hostname -i
```

## Check Network Interfaces

```bash
ip a
```

## Test Network Connectivity

```bash
ping 8.8.8.8
```

## Check Running Services

```bash
systemctl list-units --type=service
```

## View OS Information

```bash
cat /etc/os-release
```
