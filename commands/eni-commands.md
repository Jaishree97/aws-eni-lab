# ENI Commands

## Display Network Interfaces

```bash
ip a
```

## Show Host IP Address

```bash
hostname -i
```

## Check Routing Table

```bash
ip route
```

## Display Interface Details

```bash
ip addr show
```

## Verify Internet Connectivity

```bash
ping google.com
```

## Check Listening Ports

```bash
ss -tulnp
```

## Verify Apache Port

```bash
netstat -tulnp
```
